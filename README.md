# 🐾 Patinhas do Coração

## 📖 Sobre o Projeto

O **Patinhas do Coração** é um site desenvolvido para facilitar a adoção responsável de animais. A plataforma apresenta uma lista de pets disponíveis para adoção, permitindo que os usuários visualizem informações sobre cada animal e enviem uma solicitação de interesse por meio de um formulário simples e intuitivo.

O projeto foi criado com o objetivo de praticar conceitos de desenvolvimento web utilizando HTML, CSS e JavaScript, além de conscientizar sobre a importância da adoção responsável.

---

## 🚀 Funcionalidades

* Visualização de pets disponíveis para adoção.
* Exibição de informações básicas de cada animal.
* Formulário de solicitação de adoção.
* Interface simples e responsiva.
* Mensagem de confirmação após o envio da solicitação.
